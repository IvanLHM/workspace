# Spring Boot 权限管理系统

这是一个基于 Spring Boot 和 AdminLTE 的权限管理系统。

## 技术栈

- Spring Boot 3.1.5
- Spring Security
- MyBatis
- PostgreSQL
- Thymeleaf
- AdminLTE 3.2

## 功能特性

- 用户认证和授权
- 基于角色的访问控制（RBAC）
- 角色管理
- 权限管理
- 响应式界面

## 快速开始

1. 克隆项目
```bash
git clone [你的仓库URL]
```

2. 使用 Docker Compose 启动项目
```bash
docker-compose up -d
```

3. 访问系统
- URL: http://localhost:8080
- 默认管理员账号: admin
- 默认密码: admin123

## 开发环境要求

- JDK 17+
- Maven 3.6+
- Docker & Docker Compose
- PostgreSQL 14+
